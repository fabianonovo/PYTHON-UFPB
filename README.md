# PYTHON-UFPB

Repository of Python exercises from UFPB courses. Organized by topics and difficulty levels.

## Repository Structure

### Implemented Modules
- **`introduction-to-programming/`**: Basic syntax, input/output, and variable manipulation.
- **`conditional-structures/`**: Decision-making with if/else statements and logical operators.

## Technologies
- **Python 3**

## Notes
- Some files under `conditional-structures/` may contain bugs or simpler implementations due to the author's learning order. At the time these exercises were written the author hadn't learned other topics yet (for example: functions, tuples, exception handling with `try/except`, and more). These are intentional artifacts of learning — feel free to improve them as you advance.
