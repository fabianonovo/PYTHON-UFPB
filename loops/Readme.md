# Loops (Repetition Structures)

This folder contains exercises focused on repetition flow control in Python, developed during my studies at UFPB.

## Concepts Covered

* Sequential execution with loops
* Pre-test loops (`while`)
* Post-test loops (`do-while` patterns)
* Counter-controlled loops (`for`)
* Accumulator patterns
* Loop control statements (`break`, `continue`)
* Nested loops
