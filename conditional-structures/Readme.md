# Conditional Structures

This folder contains exercises focused on decision-making flow control in Python, developed during my studies at UFPB.

#Concepts Covered
* Short-circuit evaluation with logical operators (`and`, `or`, `not`)
* Relational operators (`>`, `<`, `==`, `!=`, `>=`, `<=`)
* Single-alternative decisions (`if`)
* Dual-alternative decisions (`if-else`)
* Multiple-alternative decisions (`if-elif-else`)
