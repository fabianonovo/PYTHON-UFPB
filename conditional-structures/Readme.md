# Conditional Structures

This folder contains exercises focused on decision-making flow control in Python, developed during my studies at UFPB.

## Concepts Covered

* Short-circuit evaluation with logical operators (`and`, `or`, `not`)
* Relational operators (`>`, `<`, `==`, `!=`, `>=`, `<=`)
* Single-alternative decisions (`if`)
* Dual-alternative decisions (`if-else`)
* Multiple-alternative decisions (`if-elif-else`)

## Solved Problems

* **Algorithm_calculate_highest_value.py** - Compares two input values and determines the highest one using if-elif-else structure.

* **algorithm_calculate_salary_increase.py** - Calculates a salary increase percentage (10% or 15%) based on whether the current salary is above or below R$ 1,250 using nested conditionals.

* **algorithm_charged_per_trip.py** - Calculates the trip fare based on distance traveled, applying different rates: R$ 0.50/km for up to 200km and R$ 0.45/km beyond that.

* **algorithm_compare_show_max_speed_street.py** - Validates if a car's speed exceeds the 80 km/h limit and applies a R$ 5 fine if necessary using conditional logic.

* **algorithm_highest_lowest_value.py** - Identifies the highest and lowest values among three input numbers using multiple conditional comparisons.

* **algorithm_loan_purchasing_home.py** - Evaluates eligibility for a home purchase loan by checking if the monthly payment doesn't exceed one-third of the annual salary.

* **algorithm_to_calculate_tax.py** - Calculates income tax using tax brackets: 20% for income between R$ 1,000-R$ 3,000 and 30% for income above R$ 3,000.

* **eletricy_costs.py** - Calculates electricity bills based on installation type (residential, commercial, or industrial) and consumption in kWh with different rate tiers.

* **two_values_operation.py** - Performs basic arithmetic operations (+, -, *, /) on two input values with validation for invalid operations and division by zero.
